# Livre d'accueil numérique – test

## Fichiers
- index.html : contenu du livre d'accueil
- style.css : apparence
- script.js : bouton de copie du Wi-Fi

## Mise en ligne sur GitHub Pages
1. Créez un dépôt GitHub public.
2. Ajoutez les trois fichiers à la racine du dépôt.
3. Ouvrez Settings > Pages.
4. Dans Source, choisissez Deploy from a branch.
5. Sélectionnez la branche main et le dossier /(root).
6. Cliquez sur Save.

Le site sera disponible à une adresse du type :
https://votre-identifiant.github.io/nom-du-depot/
